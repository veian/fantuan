#饭团

可以帮助你记录你和你的好友一起出去吃饭时的开销，并且提供一些有用的排行榜。
这是一个简单的例子用来展示一个基于Java的HTML5企业级应用架构

###开发环境
* Eclipse 3.7 J2EE
* AspectJ Development Tools 2.2

###使用的框架
####客户端
* AngularJS
* Bootstrap
* JQuery
* Angular-UI
* select2
* font-awesome

####服务器端
* Spring
* JAX-RS - Resteasy
* JPA - EclipseLink
* Orika - DTO/BO Mapping Library
