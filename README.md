#饭团

可以帮助你记录你和你的好友一起出去吃饭时的开销，并且提供一些有用的排行榜。
这是一个简单的例子用来展示一个基于Java的HTML5企业级应用架构

###开发环境
* Eclipse 3.7 J2EE
* AspectJ Development Tools 2.2
* Maven 3
* NodeJS 0.10
* Brunch 1.7
* Bower 1.2
* CoffeeScript 1.6

###使用的框架
####客户端
* AngularJS
* Bootstrap
* JQuery

####服务器端
* Spring Framework
* JAX-RS - Resteasy
* JPA - EclipseLink
* Spring Security
* Orika - DTO/BO Mapping Library
